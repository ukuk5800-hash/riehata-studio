## デザイン原則

0. **管理操作はチャット主導** — GUIは補助、自然文実行を最短導線に
1. **予約体験の分かりやすさ** — 非ITユーザーでも迷わない
2. **余白を十分に** — 文字サイズ・行間・パディングを広めに
3. **カラー一貫性** — 顧客: teal / 管理: indigo / 警告: amber / エラー: red
4. **レスポンシブ** — スマートフォンでも快適に予約できる
5. **多言語** — ハードコードせず messages/ から取得
